问题背景📑

  1. Jetbrains中使用Markdown文件时
  2. 按ctrl+alt+l格式化代码
  3. Markdown中的代码块中的代码格式会乱

痛点分析📑

Jetbrains中的Markdown插件还不够完善

  1. 按ctrl+alt+l格式化代码会将代码块中的代码格式变乱

解决方案📑

禁用代码块中的编码辅助

  1. 👇解决方案来自Jetbrains官方文档 https://www.jetbrains.com/help/idea/markdown.html#disable-injection-in-code-blocks
  2. If your code blocks are not meant to be syntactically correct, you may want to disable code injection and syntax errors in code blocks.
  3. 如果代码块的语法不正确，则可能希望禁用代码块中的代码注入和语法错误。
  4. 👇打开IDE设置，然后选择Languages & Frameworks | Markdown。
  5. IDE左上角⇨File⇨Settings⇨Languages & Frameworks⇨Markdown
  6. 👇清除以下选项(将前面的√去掉)
  7. Inject languages in code fences
  8. Show problems in code fences
  9. 👇Click OK to apply the changes.
  10. 点击OK或Apply即可

参考

  • [1 ] Jetbrains官方文档 https://www.jetbrains.com/help/idea/markdown.html#disable-injection-in-code-blocks
